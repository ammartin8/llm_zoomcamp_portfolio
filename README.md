# 🧠 LLM Zoomcamp Course & Portfolio Capstone

> Building production-ready AI assistants with Retrieval-Augmented Generation (RAG), agents, vector search, and more.

Welcome to my public repository showcasing the comprehensive **LLM Zoomcamp** I am currently taking from [DataTalks.Club](https://datatalks.club/). This project encapsulates my journey through a rigorous curriculum designed to bridge the gap between LLM theory and production-grade AI applications.

By completing all modules of this 10-week course, I will learn how to build an AI system that answers questions about custom knowledge bases, implement agentic workflows, master vector search and embeddings, design evaluation frameworks, and deploy monitored LLM applications in production.

---

## 📚 What I Will Learn

Throughout this intensive program, I will move beyond basic LLM interactions to architecting full-stack AI systems. Here is a breakdown of the core competencies I am developing:

- **LLM Fundamentals & RAG:** Master what Large Language Models are, why they need external context, and build Retrieval-Augmented Generation pipelines from scratch
- **Vector Search & Embeddings:** Learn how to embed text into vector representations, use vector databases for semantic search, and implement hybrid search strategies
- **Agentic Workflows:** Design AI agents that can plan, use tools, call functions, and execute multi-step tasks autonomously
- **Prompt Engineering & Construction:** Master prompt design patterns, few-shot learning, chain-of-thought reasoning, and instruction tuning techniques
- **Evaluation & Monitoring:** Build evaluation frameworks to assess model performance, detect drift, monitor latency/costs, and ensure production reliability
- **Production Deployment:** Learn best practices for deploying LLM applications with proper error handling, rate limiting, caching, and observability
